Albedo

Альбедо играбельный Гео персонаж в Genshin Impact.

Гений, именуемый Принцем мела. Главный алхимик и глава исследовательской команды Ордо Фавониус.

«Гений», «Принц мела», «Старший исследователь»... Титулы и слава не значат для него ничего, когда кругом столько объектов для исследований. Ему не нужны ни богатства, ни связи. Его стремления подчинены одной цели - заполучить сокровенные знания всех поколений учёных.

«Хочешь изучить происхождение мира? Значит, нам с тобой по пути.»
